this package contains a series of scripts that allow you to create objects procedurally with specific patterns in editor and in runtime.
features<br></br>
editor wizards and commands to place objects in rectangular,cubical,circular and spherical patterns or on the surface below them.
a tile editor window that allowss you to quickly create a tileset from objects using checkboxes.
An editor window that allows you to create a tile set using an image file (e.g. a maze level drawing).
wizards to place a set of objects randomly inside a sphere/cube.
a command to create prefab from selected object.
complete documentation.
